# Titanic EDA Project

This project performs data cleaning and exploratory data analysis (EDA) on the Titanic dataset from Kaggle.

## Steps Covered
- Data Loading
- Missing Value Handling
- Feature Engineering
- Univariate & Bivariate Analysis
- Correlation Heatmap
- Key Insights

## Technologies Used
- Python
- Pandas
- Seaborn
- Matplotlib
